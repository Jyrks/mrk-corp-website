# MRK Corp Website

Professional website for MRK Corp - Full-Cycle Shipbuilding & Industrial Contracting company based in Tallinn, Estonia.

## About MRK Corp

MRK Corp is an international shipbuilding and industrial contracting company delivering turnkey teams and full-cycle solutions for shipyards and heavy-industry facilities across the globe.

## Website Structure

- **index.html** - Main homepage with services, projects, about, and contact sections
- **about.html** - Detailed about page with company information and work process
- **faqs.html** - FAQ page (not currently in navigation)

## Contact Form Setup

The contact form is configured to use **FormSpree** for handling form submissions.

**⚠️ IMPORTANT:** You need to complete the FormSpree setup before the contact form will work.

See **[FORMSPREE_SETUP.md](FORMSPREE_SETUP.md)** for detailed setup instructions.

Quick steps:
1. Sign up at [formspree.io](https://formspree.io) (free)
2. Create a new form for business@mrkcorp.io
3. Copy your Form ID
4. Replace `YOUR_FORM_ID` in index.html (line 282) with your actual Form ID
5. Upload website and test the form
